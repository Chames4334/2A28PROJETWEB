# CodeWave

## 📌 Overview

CodeWave is an intelligent insurance and leave management platform developed as part of an engineering integrated project at Esprit School of Engineering.

The platform aims to modernize HR and insurance management by integrating smart automation, AI-powered analysis, and interactive management tools to improve productivity and decision-making.

---

## 🚀 Features

### 🔹 Leave Management Module

* Global Absence Calendar
* Smart Leave Planning Assistant
* Automated Leave Validation & Rejection
* AI-based Monthly Analysis
* Employee Leave Tracking

### 🔹 User Management Module

* Secure Google Authentication
* Password Verification & Recovery via Email
* CAPTCHA Security Integration
* Intelligent AI Chatbot Assistance
* Role & Permission Management

### 🔹 Insurance Services

* Insurance Request Management
* Smart User Interaction
* Automated Workflows

---

## 🤖 AI Features

CodeWave integrates intelligent functionalities to automate HR decisions and improve user experience.

### Example:

> “Leave request rejected automatically: insufficient balance (5 days requested, current balance: 0).”

The system analyzes leave balances and automatically validates or rejects requests based on predefined rules.
